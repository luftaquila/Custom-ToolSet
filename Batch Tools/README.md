### Toolset Guide
##### 1. Brightness Controller.bat  
1. Command
```Batchfile
br [value]
```
2. Parameters  
`[value]` : Between 1 ~ 10.  
Actual value to be : `[value]` * 10 (%)
<br>

##### 2. Governor Controller.bat
1. Command
```Batchfile
gov [param]
```
2. Parameters  
`[param]` : `pf` for Performance, `bl` for Balanced, `sv` for Powersave
<br>

##### 3. Power Controller.bat
1. Command
```Batchfile
pwr [param]
```
2. Parameters  
`[param]` : `hbr` to Hibernate, `rst` to Reboot, `sdn` to Shutdown
<br>

##### 4. Mute Toggler.bat
1. Command
```Batchfile
mute
```

##### 5. Server Starter.bat
1. Command
```Batchfile
localhost
```

##### 6. Battery Reporter.bat
1. Command
```Batchfile
batteryreport
```
