# KeySet

AutoHotKey Custom KeySet Configuration

1. **`LShift + Space`** : 한영 전환
1. **`RShift`** : BackSpace

1. **`한/영`** : cmd 실행
1. **`Ctrl + Shift + ~`** : Pross Explorer 실행

1. **`한자`** : 
	+ 1회 : 붙여넣기
	+ 2회 : 복사
	+ 3회 : 잘라내기
	+ 그 외 : 우클릭

1. **`CapsLock + J`** : ←
1. **`CapsLock + L`** : →
1. **`CapsLock + I`** : ↑
1. **`CapsLock + K`** : ↓

1. **`~ + J`** : Home
1. **`~ + L`** : End

1. **`좌클릭 + 우클릭`** : 앞으로 가기
1. **`우클릭 + 좌클릭`** : 뒤로 가기

1. **`Greek Alphabet name + Tab`** : Greek Alphabet

1. **`@ + Tab`** : E-mail
1. **`wheart + Tab`** : ♡
1. **`bheart + Tab`** : ♥
1. **`wstar + Tab`** : ☆
1. **`bstar + Tab`** : ★
1. **`tld + Tab`** : ~
1. **`ud + Tab`** : _
